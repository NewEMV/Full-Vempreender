# **App Name**: Vempreender AI

## Core Features:

- Intelligent Chatbot: Simulates personalized conversations with new customers, leveraging your knowledge to answer questions, explain services, and qualify leads using AI reasoning to respond and incorporate key information.
- Revamped Landing Page: Modernized, visually appealing landing page with a dark theme, clean design, and interactive elements.
- FAQ Accordion: Accordion-style FAQ section addressing common customer queries.
- Automated lead collection: Gather leads via the AI chatbot
- Referral program: tracks when a user comes from a referral, so that they receive appropriate pricing.

## Style Guidelines:

- Primary color: Gold (#F5B25C) to convey sophistication and value.
- Background color: Dark navy (#121826) to create a modern and sophisticated dark theme.
- Accent color: Teal (#03A6A6) to highlight key interactive elements.
- Body and headline font: 'Genos', a versatile sans-serif (Note: currently only Google Fonts are supported).
- Use of glassmorphism effects for a modern, clean interface, with cards and buttons featuring rounded corners and subtle shadows.
- Subtle hover animations on buttons and interactive elements to enhance user experience.
